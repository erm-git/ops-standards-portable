## Core docs (required)

- `AGENTS.md` — agent guardrails (repo root)
- `README.md` — repo entrypoint (repo root)
- `CHANGELOG.md` — notable changes (repo root)
- `docs/index.md` — docs hub
- `docs/current-state.md` — “as-is” truth
- `docs/roadmap.md` — “to-be” near-term plan
- `docs/active-work.md` — WIP anchor (optional)

## One-line rule to stop drift

- If it’s **today’s truth** → `docs/current-state.md`
- If it’s **next work** → `docs/roadmap.md`
- If it’s **what you’re doing right now** → `docs/active-work.md`
- If it’s **how to write docs** → `docs/authoring-standards/`
