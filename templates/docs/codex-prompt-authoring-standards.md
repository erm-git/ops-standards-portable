## Purpose

Write prompts that are easy for an agent to execute safely and repeatably.

## Portable baseline

- State the goal in one sentence.
- Provide explicit scope (files/dirs to touch).
- State how to validate (commands/tests).
- Prefer checklists over prose.

## Example (pattern)

- Goal: …
- Scope: …
- Constraints: …
- Validation: …
